# Ali Murat Gultekin — Portfolio

Personal portfolio website showcasing my work in cybersecurity, machine learning, and software development.

**Live site:** [alimuratgultekin.com](https://alimuratgultekin.com)

## Tech Stack

- **Framework:** Next.js (App Router, TypeScript)
- **3D Graphics:** React Three Fiber + custom GLSL shaders
- **Animations:** Framer Motion
- **Styling:** Tailwind CSS

## Features

- Interactive 3D particle sphere with mouse-reactive repulsion and heat effects
- Scroll-triggered section animations with staggered reveals
- Simple Mode toggle for reduced motion and accessibility
- Fully responsive dark theme
- SEO optimized with Open Graph and Twitter Card metadata

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view locally.

## Deployment

Deployed on [Vercel](https://vercel.com) with automatic deploys from the `main` branch.
